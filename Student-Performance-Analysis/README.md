# 🎓 Student Performance Analysis

## 📊 Project Overview
Analyzed 500 student records to identify key factors 
affecting academic performance.

## 🛠️ Tools Used
- Microsoft Excel (Pivot Tables, Charts, Dashboard)

## 📌 Key Findings
- Students studying 26-30 hrs/week score avg 73.8 vs 39.1 for 1-5 hrs
- High School parent education has highest pass rate at 70.5%
- Students who pass have avg attendance of 77.9% vs 73.5% who fail
- Overall pass rate: 64.6% (323 out of 500 students)
